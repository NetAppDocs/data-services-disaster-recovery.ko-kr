= NetApp Disaster Recovery 개요 사용
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/use/


[role="lead"]
NetApp Disaster Recovery 사용하면 다음과 같은 목표를 달성할 수 있습니다.

* link:../use/dashboard-view.html["재해 복구 계획의 상태를 확인하세요"]
* link:../use/sites-add.html["사이트에 리소스 추가"]
* link:../use/resource-group.html["리소스 그룹을 생성하여 데이터 소스 구성"]
* link:../use/drplan-create.html["재해 복구 계획 만들기"]
* link:../use/replicate.html["VMware 앱 복제"] SnapMirror 복제를 사용하여 기본 사이트에서 클라우드의 재해 복구 원격 사이트로
* link:../use/migrate.html["애플리케이션 마이그레이션"] 기본 사이트에서 다른 사이트로
* link:../use/failover.html["장애 조치 테스트"] 리소스를 중단하지 않고
* 재해 발생 시, link:../use/failover.html["기본 사이트를 장애 조치합니다"]
* 재해가 해결된 후 재해 복구 사이트에서 기본 사이트로 link:../use/failback.html["실패로 돌아가다"]
* 작업 모니터링 페이지에서 link:../use/monitor-jobs.html["재해 복구 작업 모니터링"]
